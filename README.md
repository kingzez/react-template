# React Template

### Use Package

- react
- react-router v5
- redux
- react-redux
- connected-react-router
- antd
- prop-types


### Usage

install dependency
```shell
npm i
```

dev
```shell
npm start
```

build
```shell
npm run build
```

analyze build
```shell
npm run analyze
```

serve static, need build first
```shell
npm run serve
```
